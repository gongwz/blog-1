选项 描述
REPOSITORY 所属仓库
TAG 标签
IMAGE ID 进项ID
CREATED 创建时间
SIZE 镜像创建时间

容器操作

  • 启动一个新容器
  • 启动一个已经存在的容器
  • 启动一个容器退出时不关闭
参数 描述
–name 镜像的名字
-t 分配一个伪终端
-i 容器标准输入保持打开状态

通过容器的ID我们可以发现和容器里面的主机名是一致的

  • 启动一个容器在退出的时候自动删除这个容器
  • 进入一个已经运行的容器

通过nsenter访问容器且推出时不销毁容器

nsenter访问的是一个进程的命名空间

  • 查看所有的容器
选项 描述
CONTAINER ID 容器的ID
IMAGE 通过哪个镜像来运行的
COMMAND 最后执行的命令
CREATED 创建时间
STATUS 状态
PORTS 启动的端口
NAMES 容器的名称，名称默认是自动生成的
  • 删除一个容器
  • 删除一个正在运行的容器
  • 停止所有正在运行的容器
  • 删除所有容器
  • 删除状态为退出的所有容器
