Python数据结构与算法系列之选择排序

创建日期：

选择排序（Selection sort）是一种简单直观的排序算法。它的工作原理如下：

  1. 首先在未排序序列中找到最小（大）元素，存放到排序序列的起始位置；
  2. 然后，再从剩余未排序元素中继续寻找最小（大）元素；
  3. 然后放到已排序序列的末尾；
  4. 以此类推，直到所有元素均排序完毕；
